Тестовое задание

Проект выполнен на Vite + HTML + SCSS.

Используемые технологии

- HTML5
- SCSS
- JavaScript
- Vite

Запуск проекта

Установить зависимости

```bash
npm install
```

Запустить локальный сервер

```bash
npm run dev
```

Собрать production-версию

```bash
npm run build
```

Предпросмотр production-сборки

```bash
npm run preview
```
